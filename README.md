## Corn Leaf Disease Detection - UI/UX Design

## Overview
This repository contains the UI/UX design for an AI-powered Corn Leaf Disease Detection platform designed to help farmers identify corn leaf diseases through image recognition.

## Tools Used
- Figma
- User-Centered Design Principles

## Features
- Disease Detection Interface
- Farmer Dashboard
- Image Upload Screen
- Disease Analysis Results
- Responsive Mobile Design
